## 二 程序的架构设计
### 1、程序设计的好处
    1）思路清晰
    2）不会出现写一半代码时推翻重写
    3）方便自己或以后的同事维护
### 2、三层架构设计的好处
    1）把每个功能都分成三部分，逻辑清晰
    2）如果用户更换不同的用户界面或不同的数据存储机制都不会影响接口层的核心逻辑代码，扩展性强
    3）可以在接口层，准确的记录日志和流水。
